# Civilization

Браузерна покрокова стратегія в дусі Civilization 1 (1991).
Next.js + React + Canvas. Деплой: Vercel.

## Запуск локально
```bash
npm install
npm run dev
```
Відкрий http://localhost:3000

## Можливості
- Процедурна карта: річки, узбережжя, ресурси, хатини племен
- 10 юнітів, флот із перевезенням десанту, дипломат і найманці-варвари
- 16 технологій, будівлі, 5 чудес світу
- Уряди (деспотизм/монархія/республіка), податки, щастя і заворушення
- Дипломатія, варвари, нові цивілізації на руїнах міст
- Маршрути (GoTo), події ходу, збереження гри (localStorage)
